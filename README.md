## 🚀 Overview

**DODO Studio** is a professional-grade, browser-based design application that bridges the gap between freeform ideation (**Mood Board**) and structured execution (**Designer**). Built with a focus on speed, precision, and high-fidelity aesthetics, it offers a dual-mode experience tailored for designers, artists, and creators.

### [✨ 🔗 Experience the Live App here](https://vibecode-phi-ivory.vercel.app/)

---

## 💎 Core Features

### 🎨 Dual-Mode Canvas
- **Mood Board Mode**: Freeform, hand-drawn aesthetic powered by [Rough.js](https://roughjs.com/). Perfect for sketching, brainstorming, and lo-fi wireframing.
- **Designer Mode**: Pixel-perfect, high-fidelity artboard environment with grid snapping and professional depth.

### 🛠 Professional Toolset
- **Geometry & Paths**: Rectangles, Circles, Lines, Arrows, and a flexible Hand-drawn Pencil.
- **Advanced Text Tool**: Excalidraw-style text editing with auto-resizing and intuitive interaction.
- **Asset Import**: Seamlessly bring in **Images (PNG, JPG)** and **SVGs** to enrich your designs.

### 📦 Layer Management
- **Full-Featured Layers Panel**: Reorder with drag-and-drop, toggle visibility, lock layers to prevent accidental edits, and search through your hierarchy.
- **Atomic Operations**: Duplicate elements, group-level transformations, and unified rotation logic.

### 📐 Precision Transformations
- **Multi-Selection**: Lasso-select multiple elements and transform them as a single unit.
- **Proportional Resizing**: Maintain aspect ratios during group scaling.
- **Center-Point Rotation**: Unified rotation around the visual centroid of selected groups.

### 🌙 High-Fidelity UI
- **Glassmorphism Design**: A stunning, iOS-inspired dark/light theme with real-time blur and premium iconography.
- **Smart Property Bar**: A dynamic interface that shows only the controls relevant to your current selection.

| Technology | Purpose |
| :--- | :--- |
| **React + Vite** | High-performance frontend foundation |
| **Rough.js** | Hand-drawn canvas engine |

## 🏗 Installation

Experience the power of DODO Studio locally:

```bash
# Clone the repository
git clone https://github.com/subhoS/dodo-studio.git

# Navigate to directory
cd dodo-studio

# Install dependencies
npm install

# Start the development server
npm run dev
```

---

## 🔐 Privacy by Design

DODO Studio is privacy-first. All your design data is stored locally in your browser's `localStorage`. No tracking, no cloud storage, and no data leaves your machine unless you export it.
